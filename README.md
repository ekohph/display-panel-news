# Display Panel News

Korean Markdown briefings on public news about display panel makers, panel buyers, platform ecosystem companies, and selected display supply-chain companies.

## Scope

This repository tracks public, source-linked updates related to:

- Display panel makers: LG Display, Samsung Display, BOE, TCL CSOT, Tianma, Everdisplay, Visionox, and others
- Panel buyers: Apple, Samsung Electronics, LG Electronics, Hyundai Motor Group / Kia, Meta, Valve, and other display-related product companies
- Platform ecosystem: Intel, NVIDIA, and other companies whose platform requirements affect display specifications or roadmaps
- Supply chain: display driver ICs, Micro-OLED backplanes, materials, equipment, regulation, and related suppliers

## Contents

- `summaries/<월>/`: daily display panel news summaries grouped by month, for example `summaries/5월/`
- `trends/weekly/`: weekly trend summaries

## Format

Daily summaries are written in Korean and generally use this structure:

1. `핵심 요약`
2. `패널 메이커 신규 기사`
3. `Panel buyers 동향`
4. `Platform ecosystem`
5. Display supply-chain or regulation sections when relevant

Article bullets include source links directly below the bullet:

```md
- Article summary.
  원문: https://...
  교차 확인: https://...
```

`교차 확인` is included only when a useful secondary source is available.

## Notes

- The summaries are based on public web sources.
- Repeated articles or substantially identical storylines are not intended to be repeated as new items.
- Company coverage is based on news relevance, not endorsement or affiliation.
